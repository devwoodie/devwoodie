## Quick Start
How to run `devwoodie CLI` by [**NPM**](https://www.npmjs.com/package/devwoodie)
```bash
npx devwoodie
```
```bash

    -g, --github     Open Github
    -b, --blog       Open Blog
    -p, --portfolio  Open Portfolio
    -r, --resume     Open Resume
    -h, --help       display help for command
```
